# I18n 国际化架构

## 概述

ChatAIO 实现了完整的国际化(i18n)功能，支持 5 种语言：
- en-US (English) - 源语言
- zh-CN (简体中文)
- zh-TW (正體中文)
- ja-JP (日本語)
- ko-KR (한국어)

## 架构设计

### 单一数据源

```
user-settings.json → appearance.language (唯一的语言持久化位置)
```

Electron 模式下禁用 localStorage 持久化，统一以 `user-settings.json` 为单一数据源。

### 进程隔离

- Renderer 和 Main 各自维护独立的 i18n 实例
- 通过 IPC 同步语言设置
- 避免直接共享状态


## 启动流程

```
┌─────────────────────────────────────────────────────────┐
│ Main Process                                            │
│                                                         │
│ 1. reaxel_I18n 初始化                                   │
│    └→ 从 user-settings.json 读取 appearance.language    │
│    └→ 加载对应语言资源 (languageMaps)                    │
│                                                         │
│ 2. setI18nInstance(reaxel_I18n) 注入到 Menu / Tray      │
│    └→ Menu.createMenu() 中 t() 使用正确语言翻译         │
│                                                         │
│ 3. 注册 'language-change' IPC handler                   │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│ Renderer Process (SettingsView)                         │
│                                                         │
│ 1. reaxel_I18n 初始化 (Electron模式不用localStorage)     │
│    └→ 默认 sourceLanguage: en-US                        │
│                                                         │
│ 2. reaxel_SettingsView → loadSettingsOnStartup()        │
│    └→ IPC: fetch-settings → 获取 settings               │
│    └→ setSettings() 中同步: reaxel_I18n().setLanguage() │
└─────────────────────────────────────────────────────────┘
```

## 运行时语言切换流程

```
用户在 Appearance 面板选择新语言
         │
         ├─→ ① setState({ language: value })  // 更新 UI 控件状态
         │
         ├─→ ② reaxel_I18n().setLanguage(value)  // 渲染进程 i18n 立即切换
         │
         └─→ ③ api.languageChange(value)  // IPC 通知主进程
                    │
                    ↓
         Main: useIpcRendererToMain('language-change').on(...)
                    │
                    ├─→ reaxel_I18n().setLanguage(lang)  // 主进程 i18n 切换
                    ├─→ reaxel_Menu().rebuildMenu()       // 重建菜单
                    └─→ updateTrayMenu()                  // 更新托盘菜单
```

## Apply/Save 持久化流程

```
用户点击 Apply
    │
    └→ applySettings(buildSettingsFromStore())
         │
         └→ Main: settingsConfigService.saveSettings(...)
              └→ 写入 user-settings.json（下次启动时生效）
              └→ reaxel_I18n().setLanguage(...)  // 防御性同步
```

## IPC 通信

### IpcSchema 定义

```typescript
'language-change': IpcStructure.RendererToMainEvent<[language: string], {channel:void, args:void[]}>;
```

### 主进程监听

```typescript
useIpcRendererToMain('language-change').on((e, language) => {
	reaxel_I18n().setLanguage(language as any);
	reaxel_Menu().rebuildMenu();
	if (isTrayActive()) {
		updateTrayMenu();
	}
});
```

## 关键规则（Reaxel 使用模式）

### setI18nInstance 传参规则

Menu/Tray 的 `t()` 需要 `i18nInstance().i18n(text)` 模式，必须传递 **reaxel 引用**（可调用），而非 `reaxel()` 的返回值：

```typescript
// ✅ 正确 - reaxel_I18n 本身是可调用的
reaxel_Menu().setI18nInstance(reaxel_I18n);

// ❌ 错误 - reaxel_I18n() 返回 rtn (普通对象)，不可再调用
reaxel_Menu().setI18nInstance(reaxel_I18n());
```

## 翻译覆盖范围

### 主菜单 (Main Menu)
- Application / Settings / Check for Updates
- View / Reload / Force Reload / Developer Tools / Zoom In
- Switch AI / No enabled AI pages

### 托盘菜单 (Tray Menu)
- Show Window / Quit

### SettingsView
- 侧边栏菜单：Networks, Appearance, Manage AIs, System
- 底部按钮：Exit Without Submit, Discard All Changes, Apply, Save All
- 对话框：Restart required 等
